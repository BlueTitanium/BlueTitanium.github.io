Tic-Tac-Toe with AI

This is the first project that I ever made. It was an assignment from my APCS teacher at Bronx Science, but the assignment was just to create Tic-Tac-Toe with random AI. However, I really wanted to challenge myself, so I made three versions: Player Vs. Player Tic-Tac-Toe, Tic-Tac-Toe with the MiniMax Algorithm, and Tic-Tac-Toe with the Alpha-Beta Pruning Algorithm. Overall, I really enjoyed coding this project because, before that, I had this weird notion that algorithm design was difficult and hard to implement. While coding this, I discovered that it wasn't hard, as long as I broke everything up into steps. That mindset helped me greatly in future projects.

Calligrider

This was my first ever hackathon project. The concept was a fighter game that relies on momentum and collision calculations all in JavaScript. I coded this with my friend, Pietro Saia. Something that I wasn't used to while programming this project, was the massive time constraint. We basically had 8 hours to code the entire thing. Another issue was that I'd never worked with JavaScript and HTML before this project. However, despite all of the issues, we ended up with a playable game, and considering the constraints that we worked with, I'm proud of the final product. The work on this project prepared me well for future hackathons and projects where I had a tough time limit.
